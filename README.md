# Atulya HR Suite

Free HR and payroll workflow suite for Indian businesses — from joining to full & final settlement.

## Features

- Employee master database
- Attendance tracking (biometric/Excel import)
- Leave management (EL, CL, SL, LWP)
- Payroll processing with PT, PF, ESI, TDS
- Salary slip generation (PDF)
- Full & final settlement calculator
- Statutory register generation
- Expense & reimbursement management

## Quick Start

```bash
pip install atulya-hr-suite
atulya-hr init
atulya-hr payroll generate --month 2026-04
atulya-hr reports pf --month 2026-04
```

## Modules

| Module | Description |
|--------|-------------|
| Employees | Master data & documents |
| Attendance | Daily logs, monthly summary |
| Leave | Apply, approve, balance tracker |
| Payroll | Salary calc, PT, PF, ESI, TDS |
| Reports | Salary slips, registers, returns |
